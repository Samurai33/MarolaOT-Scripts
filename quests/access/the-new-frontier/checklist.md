# Checklist assistido — The New Frontier

Este checklist não substitui o Quest Log. Antes de cada etapa, confirme manualmente o estado exibido no cliente.

## Preparação

- [ ] confirmar que o personagem consegue viajar para Farmine;
- [ ] levar dinheiro para viagens;
- [ ] confirmar acesso a Kazordoon, Edron, Port Hope, Thais e Venore;
- [ ] manter CaveBot, TargetBot, AttackBot e HealBot desligados;
- [ ] não iniciar nenhuma arena sem supplies e rota de retorno;
- [ ] ter outro jogador level 70+ disponível para a missão 9.

## Missão 1 — New Land

- [ ] falar manualmente com Ongulf para iniciar;
- [ ] capturar rota até o elevador;
- [ ] parar antes da alavanca;
- [ ] usar elevador manualmente;
- [ ] capturar passagem da montanha;
- [ ] confirmar atualização do Quest Log;
- [ ] retornar e reportar manualmente.
