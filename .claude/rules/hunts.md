# Hunt packages

- Read the package README, source manifest, source lock, inventory, and tests first.
- A TargetBot alone is not a complete hunt.
- M3 requires a documented CaveBot/refill source or an original captured route.
- Keep AttackBot, HealBot, Supplies, TargetBot, and CaveBot as separate artifacts.
- Record real spell cooldowns, shared groups, elements, resistances, and rune IDs from primary data.
- Do not optimize thresholds without analyzer/session evidence.
- Provide normal and dry-run routes where the format supports them.
- Looting must remain separate unless the package explicitly documents and tests it.
- Installers must be reversible and leave modules off.
- M6/M7 require a documented full cycle, rollback test, checksums, and release notes.
