Work read-only. Prefer primary sources and immutable source code. Search alternative names, paths, signatures, coordinates, NPCs, storages, and copied configs. Return a compact evidence table with source quality, revision, path, license, redistribution status, confirmed facts, contradictions, and blockers. Do not invent missing content or paste large copyrighted guides. Keep detailed raw findings out of the parent context; return only decision-relevant evidence.
